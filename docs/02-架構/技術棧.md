# 技術棧

> 版本以各 `*.csproj` 的 `PackageReference` 為準（撰寫時的實際值）。要確認最新版本請直接看對應 `.csproj`。

## 1. 平臺與語言

- **語言**：C#（`Nullable` enable、`ImplicitUsings` enable）。
- **目標框架**：.NET 9（`net9.0`），全部 9 個專案一致。
- **Web 框架**：ASP.NET Core 9 + **Blazor Server（Razor Components，互動式伺服器渲染）**。

## 3. 資料存取

- **ORM**：Entity Framework Core 9.0.3（`Microsoft.EntityFrameworkCore.Sqlite`）。
- **資料庫**：SQLite（`BackendDB.db`）為主；另支援 SQL Server LocalDB 連線字串。
- **DbContext**：`BackendDBContext`（`CTMS.EntityModel`）。
- 啟動時自動執行 `Database.Migrate()` 並建立預設專案/角色/管理者帳號。

詳見 [06-部署與維護/資料庫與EFCore.md](../06-部署與維護/資料庫與EFCore.md)。

## 8. 執行環境傾向

專案明顯偏向 **Windows**：

- `Syncfusion.HtmlToPdfConverter.Net.Windows`（PDF 轉換 Windows 專用）。
- 預設檔案路徑為 `C:\...`（DICOM、Queue、Inference、風險評估模型）。
- DICOM 與檔案處理流程以 Windows 路徑為前提。

部署到 Linux 前需重新檢查 PDF 轉換、檔案路徑與影像處理依賴。
