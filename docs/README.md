# CTMS 文件總導覽

本目錄是 CTMS（婦科癌症臨床試驗管理平臺）的系統文件，供**人類開發者與 AI 助手**在修改或新增程式碼前，快速理解系統架構、資料模型、各功能模組的實作與商業邏輯。

> 編碼規範：本目錄所有 `.md` 採 UTF-8 **含 BOM**、繁體中文，由 [scripts/Test-DocsEncoding.ps1](../scripts/Test-DocsEncoding.ps1) 於 CI 強制檢查。

## 從哪開始

- 第一次接觸：先讀 [01-系統總覽/系統總覽.md](01-系統總覽/系統總覽.md)。
- 要改某個畫面/功能：直接看 [04-功能模組/](04-功能模組/) 下對應文件，重點在各文件的「**畫面商業邏輯**」章節（欄位驗證、條件顯示）。
- 要新增醫院：看 [07-擴充指南/新增醫院.md](07-擴充指南/新增醫院.md)。
- repository 根目錄的 [README.md](../README.md) 另有完整「路由與頁面功能地圖」。
